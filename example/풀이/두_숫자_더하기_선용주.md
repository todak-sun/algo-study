# 두 숫자를 더하는 함수를 작성하기

- 매개변수로 두 정수를 받아 그 합을 리턴하는 함수를 작성하세요.
- 함수의 이름은 add로 작성하세요

### javascript
```javascript
  const add = (a, b) => a + b;
```

### java
```java
  package io.study.algo

  public class Add  {
    public int add(int a, int b){
      return a + b;
    }
  }
```

### c
```c
  #include <stdio.h>

  int add(int a, int b){
    return (a + b);
  }
  
```
